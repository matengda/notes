```sh
ls -l a
# -rw-rw-r--. 1 sa sa 0 Oct 15 19:00 a
```



|  **-**  |    **文件类型**    |
| :-----: | :----------------: |
| **rw-** |   **所有者权限**   |
| **rw-** |   **用户组权限**   |
| **r--** | **其他用户的权限** |

# 文件类型

| **-** | **普通文件** |
| :---: | :----------: |
| **d** |  **文件夹**  |
| **c** | **字符设备** |
| **b** |  **块设备**  |
| **s** | **管道文件** |
| **l** |  **软连接**  |
